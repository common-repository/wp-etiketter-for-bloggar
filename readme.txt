== Description ==


Det här tillägget skapar länkar till Bloggar (http://www.bloggar.se/) från Wordpress-etiketter



== Installation ==



1. Ladda upp hela mappen till wp-content/plugins i din Wordpress installation

2. Aktivera den i inställningarna för Tillägg







== Inställningar ==

Dessa inställningar görs i menyn "Inställningar >> Etiketter för Bloggar"



1. Texten som visas till vänster om dina etiketter kan ändras

   rutan "Titel för Bloggaretiketter"



2. Genom att välja "Öppna Bloggarlänkar i ett nytt fönster?" kommer de

   att göra det.



3. Genom att välja "Lägg till "nofollow" till länkarnas rel atribut?" 

   kommer de genererade länkarna inte längre att räknas som fördel

   för målsidan av sökrobotar.



== Avancerat användande ==



Om du vill lägga till etikettlänkarna någon annanstans på sidan kan du 

stänga av "Inkludera etiketterna längst ner i inlägget?" och ändra

bloggens tema så att den innehåller "etiketter2bloggar_get_tags_links()"

funktionen där du vill ha den.



== Frequently Asked Questions == 



= Fungerar tillägget med alla WordPress-verisioner? =



Den här verisionen av tillägget fungerar med WordPress 2.3.0 och bättre.
